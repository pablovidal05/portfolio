# Fuentes del Portfolio

Esta carpeta debe contener los archivos de fuente:

## Monument Grotesk
- `MonumentGrotesk-Variable.woff2`
- `MonumentGrotesk-Variable.woff` (opcional, como fallback)
- `MonumentGroteskMono-Variable.woff2`
- `MonumentGroteskMono-Variable.woff` (opcional, como fallback)

## Guyot Text
- `GuyotText-Regular.woff2`
- `GuyotText-Regular.woff` (opcional, como fallback)
- `GuyotText-Italic.woff2`
- `GuyotText-Italic.woff` (opcional, como fallback)

## Cómo obtener las fuentes

- **Monument Grotesk**: Fuentes de pago disponibles en [Dinamo Type Foundry](https://www.dinamofonts.com/)
- **Guyot Text**: Busca información sobre dónde obtener esta fuente

Una vez que tengas los archivos, colócalos en esta carpeta (`public/fonts/`).

## Nota

Si aún no tienes los archivos de fuente, el sitio usará las fuentes de fallback hasta que agregues los archivos.
