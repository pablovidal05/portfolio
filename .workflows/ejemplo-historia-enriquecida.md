# Historia Enriquecida: Mas Analytics Landing Page

## Overview Visual

Landing page para homepage y digital roadmap de Mas Analytics, consultoría especializada en Datos e Inteligencia Artificial. Diseño que alinea la identidad de marca con objetivos digitales, mejorando la navegación y experiencia de usuario.

Este proyecto es de categoría "ecommerce-landings" y se enfoca en: Web Design, UI/UX, Landing Page, Product Design.

El diseño debe reflejar profesionalismo, modernidad y estar optimizado para experiencia de usuario.

## Guía de Estilo

### Paleta de Colores
Azul tecnológico (#0066CC), gris oscuro (#1A1A1A), blanco, acentos naranja (#FF6B35)

### Tipografía
Sans-serif moderna (Inter, Poppins, o similar)
- Títulos: Bold, 32-48px
- Subtítulos: Medium, 18-24px
- Cuerpo: Regular, 16px

### Espaciado
Generoso, con respiración visual. Márgenes consistentes de 24-48px.

### Elementos Visuales
- Iconos minimalistas
- Ilustraciones abstractas o relacionadas con el tema
- Sombras sutiles para profundidad
- Bordes redondeados (8-12px)
